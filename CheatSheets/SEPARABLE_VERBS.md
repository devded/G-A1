# Separable Verbs

| Infinitive | Statement |
|---|---|
| aufstehen | Ich stehe früh auf. |
| einkaufen | Wir kaufen am Samstag ein. |
| anrufen | Sie ruft den Arzt an. |
| anfangen | Der Kurs fängt um neun Uhr an. |
| mitbringen | Bringst du Brot mit? |

In a main clause, the prefix moves to the end and receives stress.
