# YouTube and Podcasts

## Recommended

- [Easy German](https://www.youtube.com/@EasyGerman) — use beginner-labelled content and the Super Easy German path.
- [DW Learn German](https://www.youtube.com/@dwlearngerman) — learner series and listening support.
- [Goethe-Institut](https://www.youtube.com/@goetheinstitut) — culture and language content; select only relevant beginner material.
- [Slow German](https://slowgerman.com/) — optional carefully spoken podcast input after basic foundations.

## Viewing protocol

1. Watch once for context without pausing.
2. Rewatch and write five useful items.
3. Select 10–20 seconds for shadowing.
4. Close subtitles and retell the message in three simple sentences.
5. Stop after the assigned segment.

A playlist is not a curriculum. Follow the weekly topic and time budget.
