# Basic Dative Case

## Learning purpose

Use this structure accurately in practical A1 communication.


### Concept

The dative often marks an indirect object and follows several high-frequency prepositions.

### Purpose

Use transport, give information to someone and describe basic locations.

### Sentence structure

```text
der → dem, das → dem, die → der, plural die → den (+ often -n on noun).
```

### Core examples

### Common mistakes

- Trying to master every dative use at once.
- Forgetting dem for both masculine and neuter.
- Using mit followed by accusative.

### English/Bengali comparison

Bengali can express similar relationships with postpositions. German uses prepositions plus article changes.
