# German A1 Grammar Library

This folder is a cumulative reference. Daily lessons introduce each structure in a controlled order; these files consolidate the complete A1 explanation, patterns, mistakes, exercises and revision tips.

## Recommended use

1. Study the assigned daily lesson first.
2. Open the linked reference only when the lesson tells you to.
3. Create personal examples.
4. Return during weekly revision.
