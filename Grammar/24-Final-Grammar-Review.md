# Final A1 Grammar Review

## Learning purpose

Use this structure accurately in practical A1 communication.


### Concept

Review uses retrieval, correction and performance rather than rereading.

### Purpose

Strengthen memory, locate weak areas and demonstrate the weekly milestone.

### Sentence structure

```text
Recall → test → correct → repeat → perform.
```

### Common mistakes

- Reading notes without testing yourself.
- Repeating the whole week instead of repairing specific gaps.
- Hiding mistakes instead of recording them.

### English/Bengali comparison

A review day is active training, not a day off and not a punishment.

### Quick grammar check

1. Copy one example and change one detail.
2. Say the new sentence aloud twice.
3. Write one sentence about your real life.
4. Cover the rule and explain it in your own words.



## Final production checklist

- [ ] I can conjugate `sein`, `haben`, regular verbs and frequent irregular verbs.
- [ ] I keep the finite verb in position 2 in a statement.
- [ ] I form yes/no and W-questions.
- [ ] I use nominative subjects and familiar accusative objects.
- [ ] I recognise and use basic dative phrases.
- [ ] I distinguish `nicht` and `kein` in familiar contexts.
- [ ] I use modal and separable verbs.
- [ ] I connect ideas with `und`, `aber`, `oder`, `denn`.
- [ ] I use common time, date and location phrases.
- [ ] I can describe a recent event with a small set of Perfekt forms.


## Guided practice

1. Read each model aloud twice.
2. Cover the model and rebuild it from the English cue.
3. Change the subject, time, place or object.
4. Write two true sentences about your own life.
5. Ask one related question and answer it aloud.

## Mini quiz

1. Write one correct statement using this topic.
2. Turn that statement into a question where possible.
3. Write one negative version.
4. Correct one likely beginner mistake.
