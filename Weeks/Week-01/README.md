## 📚 Week 1 Vocabulary Preview

You will learn **56 essential words** this week:

| Category | Examples |
|:---|:---|
| **Greetings** | Hallo, Guten Morgen, Tschüss, Auf Wiedersehen |
| **Courtesy** | Bitte, Danke, Sehr angenehm |
| **Pronouns** | ich, du, er, sie, wir, Sie |
| **Key verbs** | sein (be), heißen (called), kommen (come), wohnen (live) |
| **Countries** | Deutschland, Schweiz, Österreich |
| **Numbers** | eins–zwanzig (1–20) |
| **Nouns** | der Mann, die Frau, das Kind, die Stadt, das Land |
| **Questions** | wie, was, wer, wo, woher, wie alt |

---

## 🎓 Week 1 Grammar Topics

1. **German sounds** — ä, ö, ü, ß, ei, ie, eu, ch
2. **Verb sein** — bin, bist, ist, sind, seid, sind
3. **Regular verb endings** — -e, -st, -t, -en, -t, -en
4. **Articles der/die/das** — the 3 genders + gender hints
5. **Indefinite articles** — ein/eine
6. **W-question words** — wie, was, wer, wo, woher, wie alt, wann
