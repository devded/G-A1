## Week 1 Grammar Topics

1. **German sounds** — ä, ö, ü, ß, ei, ie, eu, ch
2. **Verb sein** — bin, bist, ist, sind, seid, sind
3. **Regular verb endings** — -e, -st, -t, -en, -t, -en
4. **Articles der/die/das** — the 3 genders + gender hints
5. **Indefinite articles** — ein/eine
6. **W-question words** — wie, was, wer, wo, woher, wie alt, wann
