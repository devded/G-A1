# Week 8 Mixed Exercises

## Vocabulary

1. Write the German for **to answer**.
2. Write the German for **answer** including the article.
3. Write the German for **task** including the article.
4. Write the German for **example** including the article.
5. Write the German for **for example**.
6. Write the German for **invitation** including the article.
7. Write the German for **wrong / incorrect**.
8. Write the German for **finished / ready / done**.

## Grammar

1. Write a correct statement using the week's first grammar topic.
2. Change it into a question where possible.
3. Write a negative version.
4. Write one sentence beginning with a time or place expression.
5. Correct: `Heute ich lerne Deutsch.`

## Translation

Translate into simple German:

1. I would like information, please.
2. I am learning German today.
3. Where is the station?
4. Please speak more slowly.
5. Write one personal sentence connected to Week 8.

## Integrated output

Complete one 60-second speaking response and one 50-word writing response for this milestone:

> Complete realistic Goethe A1 tasks confidently and independently.
