# Week 8 Grammar — Integration and Goethe A1 Exam Readiness

## Exam Listening


### Concept

A1 listening tests key information, not perfect word-for-word understanding.

### Purpose

Identify names, numbers, times, prices, places, intentions and whether a statement is true or false.

### Sentence structure

```text
Preview task → predict information → listen for key detail → choose → confirm on replay.
```

### Core examples

| German | English |
| --- | --- |
| Wann fährt der Zug? | When does the train leave? |
| Wie viel kostet das? | How much does that cost? |
| Wo treffen sie sich? | Where do they meet? |
| Richtig oder falsch? | True or false? |

### Common mistakes

- Panicking after one unknown word.
- Reading answer options only after the audio starts.
- Changing a correct answer because of a later unrelated word.

### English/Bengali comparison

Treat the task like information extraction: listen for the requested data, not every sentence detail.

### Quick grammar check

1. Copy one example and change one detail.
2. Say the new sentence aloud twice.
3. Write one sentence about your real life.
4. Cover the rule and explain it in your own words.

## Exam Reading


### Concept

A1 reading focuses on practical texts: signs, notices, advertisements, emails and forms.

### Purpose

Find the purpose and key details quickly.

### Sentence structure

```text
Identify text type → read question → scan names/numbers/keywords → confirm with surrounding words.
```

### Core examples

| German | English |
| --- | --- |
| Heute geschlossen. | Closed today. |
| Nur für Mitarbeiter. | Staff only. |
| Treffen um 18 Uhr. | Meeting at 6 p.m. |
| Bitte nicht rauchen. | Please do not smoke. |

### Common mistakes

- Translating every word before answering.
- Ignoring headings and visual layout.
- Choosing an answer because one word matches while the meaning does not.

### English/Bengali comparison

Use the same scanning skills you already use in English or Bengali notices.

### Quick grammar check

1. Copy one example and change one detail.
2. Say the new sentence aloud twice.
3. Write one sentence about your real life.
4. Cover the rule and explain it in your own words.

## Exam Writing


### Concept

A1 writing rewards completion of required content points using clear, controlled language.

### Purpose

Complete a form and write a short message about an everyday situation.

### Sentence structure

```text
Greeting → required points → question/request → closing. Use short sentences.
```

### Core examples

| German | English |
| --- | --- |
| Hallo Anna, | Hello Anna, |
| Ich komme am Freitag. | I am coming on Friday. |
| Kannst du mich abholen? | Can you pick me up? |
| Viele Grüße | Best wishes |

### Common mistakes

- Missing one content point.
- Writing advanced sentences with uncontrolled errors.
- Forgetting greeting, closing or noun capitalisation.

### English/Bengali comparison

Plan content first, then language. A simple complete message is stronger than an impressive incomplete one.

### Quick grammar check

1. Copy one example and change one detail.
2. Say the new sentence aloud twice.
3. Write one sentence about your real life.
4. Cover the rule and explain it in your own words.

## Exam Speaking


### Concept

The A1 speaking task uses predictable functions: introduction, asking/answering questions and making/responding to requests.

### Purpose

Demonstrate understandable basic interaction, not advanced conversation.

### Sentence structure

```text
Short introduction → question card → answer → request card → response.
```

### Core examples

| German | English |
| --- | --- |
| Ich heiße … | My name is … |
| Wo wohnen Sie? | Where do you live? |
| Kann ich bitte Ihren Stift haben? | May I have your pen, please? |
| Ja, natürlich. | Yes, of course. |

### Common mistakes

- Giving one-word answers when a short sentence is possible.
- Memorising one script and failing when the question changes.
- Speaking too quietly or too quickly.

### English/Bengali comparison

Clear pronunciation, task completion and interaction matter more than sounding native.

### Quick grammar check

1. Copy one example and change one detail.
2. Say the new sentence aloud twice.
3. Write one sentence about your real life.
4. Cover the rule and explain it in your own words.

## Mock


### Concept

A mock exam simulates timing, task order and pressure, then converts errors into a targeted revision plan.

### Purpose

Measure readiness and practise staying calm when information is incomplete.

### Sentence structure

```text
Timed attempt → no assistance → score → classify errors → repair → retest.
```

### Core examples

| German | English |
| --- | --- |
| Lesen Sie zuerst die Aufgabe. | Read the task first. |
| Arbeiten Sie ohne Wörterbuch. | Work without a dictionary. |
| Kontrollieren Sie am Ende. | Check at the end. |
| Notieren Sie Ihre Fehler. | Record your mistakes. |

### Common mistakes

- Pausing the test repeatedly.
- Checking answers during the attempt.
- Looking only at the total score and ignoring error types.

### English/Bengali comparison

The mock is a diagnostic tool. A difficult result is useful when it produces a clear revision plan.

### Quick grammar check

1. Copy one example and change one detail.
2. Say the new sentence aloud twice.
3. Write one sentence about your real life.
4. Cover the rule and explain it in your own words.

## Review


### Concept

Review uses retrieval, correction and performance rather than rereading.

### Purpose

Strengthen memory, locate weak areas and demonstrate the weekly milestone.

### Sentence structure

```text
Recall → test → correct → repeat → perform.
```

### Core examples

| German | English |
| --- | --- |
| Ich wiederhole ohne Notizen. | I review without notes. |
| Ich prüfe meine Antworten. | I check my answers. |
| Ich spreche eine Minute. | I speak for one minute. |
| Ich plane meine nächste Wiederholung. | I plan my next review. |

### Common mistakes

- Reading notes without testing yourself.
- Repeating the whole week instead of repairing specific gaps.
- Hiding mistakes instead of recording them.

### English/Bengali comparison

A review day is active training, not a day off and not a punishment.

### Quick grammar check

1. Copy one example and change one detail.
2. Say the new sentence aloud twice.
3. Write one sentence about your real life.
4. Cover the rule and explain it in your own words.
