# Week 2 Mixed Exercises

## Vocabulary

1. Write the German for **computer mouse** including the article.
2. Write the German for **male student** including the article.
3. Write the German for **female student** including the article.
4. Write the German for **smartphone** including the article.
5. Write the German for **laptop** including the article.
6. Write the German for **keyboard** including the article.
7. Write the German for **screen** including the article.
8. Write the German for **internet**.

## Grammar

1. Write a correct statement using the week's first grammar topic.
2. Change it into a question where possible.
3. Write a negative version.
4. Write one sentence beginning with a time or place expression.
5. Correct: `Heute ich lerne Deutsch.`

## Translation

Translate into simple German:

1. I would like information, please.
2. I am learning German today.
3. Where is the station?
4. Please speak more slowly.
5. Write one personal sentence connected to Week 2.

## Integrated output

Complete one 60-second speaking response and one 50-word writing response for this milestone:

> Describe family, work or study, possessions and a simple living space.
