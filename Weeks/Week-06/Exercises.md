# Week 6 Mixed Exercises

## Vocabulary

1. Write the German for **bird** including the article.
2. Write the German for **friendly**.
3. Write the German for **windy**.
4. Write the German for **cloudy**.
5. Write the German for **foggy**.
6. Write the German for **dry**.
7. Write the German for **humid**.
8. Write the German for **thunderstorm** including the article.

## Grammar

1. Write a correct statement using the week's first grammar topic.
2. Change it into a question where possible.
3. Write a negative version.
4. Write one sentence beginning with a time or place expression.
5. Correct: `Heute ich lerne Deutsch.`

## Translation

Translate into simple German:

1. I would like information, please.
2. I am learning German today.
3. Where is the station?
4. Please speak more slowly.
5. Write one personal sentence connected to Week 6.

## Integrated output

Complete one 60-second speaking response and one 50-word writing response for this milestone:

> Describe people, places and weather, and discuss plans, abilities and obligations.
