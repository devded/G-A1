# Goethe A1-Style Writing Tasks

These are original course-created tasks inspired by common A1 communicative functions.

## Task 1 — Form

You want to join a weekend German course. Complete fields for name, date of birth, address, phone, email, language and preferred course time.

## Task 2 — Message

Your friend invites you to a birthday party. Write:

- why you are writing,
- whether you can come,
- when you will arrive,
- what you can bring.

## Task 3 — Formal request

Write to a language school. Ask about course start, class time and price. State your current level.

Use the examination guide for timing and a self-scoring rubric.
