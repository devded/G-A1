# Forms and Personal Information

Practise these fields:

| German | Meaning |
|---|---|
| Vorname | first name |
| Familienname / Nachname | surname |
| Geburtsdatum | date of birth |
| Geburtsort | place of birth |
| Staatsangehörigkeit | nationality |
| Familienstand | marital status |
| Straße / Hausnummer | street / house number |
| Postleitzahl / Ort | postal code / city |
| Telefonnummer | telephone number |
| Unterschrift | signature |

## Task

Complete a fictional course-registration form, then spell the details aloud. Never publish real sensitive details in a public GitHub repository.
