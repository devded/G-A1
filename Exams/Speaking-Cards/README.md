# A1 Speaking Cards

Cut or display one prompt at a time.

## Topic cards

- Name
- Country
- City
- Language
- Family
- Work/study
- Daily routine
- Weekend
- Food
- Shopping
- Travel
- Weather
- Health

Ask one related question and answer your partner's question.

## Request cards

- water
- pen
- window
- repeat
- speak slowly
- ticket
- menu
- directions
- appointment
- help

Make a polite request. The partner accepts, refuses politely or asks a follow-up question.
