# German A1 Vocabulary Library

The course contains **950 vocabulary entries and practical lexical units**. Approximately 700 are active-production targets and 250 are recognition targets.

## How to learn an entry

1. Listen to the word.
2. For a noun, say the article and plural.
3. Recall the English and Bengali meanings.
4. Read the example aloud.
5. Build one true personal sentence.
6. Review on the same day and after 1, 3, 7, 14 and 30 days.

## Important rules

- Learn `der Tisch – die Tische`, not `Tisch` alone.
- Learn verbs in an example, not as isolated translations.
- Bengali meanings are concise learning aids; context determines the best natural translation.
- IPA is included where useful, but native audio remains the pronunciation reference.
