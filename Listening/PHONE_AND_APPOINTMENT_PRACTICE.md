# Phone and Appointment Practice

## Dialogue 1 — Doctor

`Praxis Dr. Klein, guten Morgen.`  
`Guten Morgen. Ich brauche bitte einen Termin.`  
`Geht Dienstag um zehn Uhr?`  
`Ja, danke. Mein Name ist Rahman.`

## Dialogue 2 — Repair request

`Entschuldigung, die Verbindung ist schlecht. Können Sie das bitte wiederholen?`  
`Ja. Der Termin ist am Freitag um halb drei.`

## Tasks

- Record both roles.
- Change the day and time.
- Spell the surname.
- Ask for repetition once.
- Write the appointment in a calendar format.
