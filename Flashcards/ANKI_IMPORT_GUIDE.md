# Anki Import Guide

1. Open Anki and create a deck named `German A1 Self-Study`.
2. Choose **Import File** and select a weekly CSV.
3. Use comma as separator and UTF-8 encoding.
4. Map column 1 to Front and column 2 to Back.
5. Map Week/Status/Frequency/PartOfSpeech to tags or extra fields.
6. Allow HTML only if you intentionally edit the cards later.
7. Review older due cards before importing the next week.

Import one weekly deck at a time. The master deck is intended for backup, audit or a fresh full-course setup, not duplicate import.
