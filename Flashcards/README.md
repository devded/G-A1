# Flashcards

CSV files contain `Front`, `Back`, `Week`, `Status`, `Frequency` and `PartOfSpeech`. They use semicolon-safe quoted CSV output and UTF-8 encoding.

## Best practice

- Say the answer before revealing it.
- For nouns, require article and plural.
- Read the example aloud.
- Mark recognition and production separately.
- Edit a difficult card rather than failing it indefinitely.
