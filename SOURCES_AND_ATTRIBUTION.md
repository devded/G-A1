# Sources and Attribution

## Course design

All explanations, lesson sequences, reading passages, listening scripts, exercises, quizzes and mock exams in this repository were generated specifically for this self-study course and edited into a consistent eight-week progression.

## Vocabulary seed data

A portion of the German–English headword seed data was derived and adapted from the open-source repository **gambolputty/german-nouns**, licensed under **Creative Commons Attribution-ShareAlike 4.0 International (CC BY-SA 4.0)**. The course filters, reorganises and enriches entries with weekly sequencing, Bengali learning aids, IPA, examples and pedagogical metadata.

Source: https://github.com/gambolputty/german-nouns  
Licence: https://creativecommons.org/licenses/by-sa/4.0/

Bengali lookup support used an open English–Bengali dictionary data file during build generation; resulting meanings were reviewed programmatically and should be interpreted as concise learning aids rather than exhaustive dictionary definitions.
