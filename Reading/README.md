# A1 Reading Course

Progression: words → signs → sentences → dialogues → messages/forms → stories/articles → exam passages.

## Reading method

1. Preview title, layout, names and numbers.
2. Read once for purpose.
3. Read the questions.
4. Scan for required information.
5. Check only useful blocking words.
6. justify each answer with a phrase from the text.
