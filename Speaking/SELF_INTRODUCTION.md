# Self-Introduction

## Core model

`Guten Tag. Ich heiße ... Ich komme aus ... und wohne jetzt in ... Ich bin ... Jahre alt. Ich spreche Bengalisch und Englisch und lerne Deutsch. Ich arbeite als ... / Ich studiere ... In meiner Freizeit ...`

## Flexible question pair

- `Wie heißen Sie?` — `Ich heiße ...`
- `Woher kommen Sie?` — `Ich komme aus ...`
- `Wo wohnen Sie?` — `Ich wohne in ...`
- `Was sind Sie von Beruf?` — `Ich bin ... von Beruf.`
- `Welche Sprachen sprechen Sie?` — `Ich spreche ...`
- `Was machen Sie gern?` — `Ich ... gern.`

Do not memorise one unchangeable speech. Practise with a different city, occupation or hobby so the patterns remain flexible.
