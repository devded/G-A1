# Common Letter Combinations


| Spelling | Sound cue | Example |
|---|---|---|
| `ei` | eye | mein |
| `ie` | long ee | sieben |
| `eu`, `äu` | oy | heute, Häuser |
| `sch` | sh | Schule |
| initial `sp` | shp | Sport |
| initial `st` | sht | Straße |
| `qu` | kv | Quelle |
| `tion` | tsyohn | Information |
| `pf` | joined p+f | Apfel |
| `ng` | one nasal sound | Wohnung |

Build contrast cards for `ei/ie`; this single visual distinction prevents many beginner errors.
