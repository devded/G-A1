# Exercise Library

Use weekly exercises first. This folder provides additional focused practice by skill and cumulative answer keys.

## Folders

- `Grammar/` — form, word order and case drills
- `Vocabulary/` — recall, classification and sentence use
- `Translation/` — controlled English/Bengali-to-German production
- `Listening/` — local audio and information extraction
- `Reading/` — graded comprehension
- `Speaking/` — prompts and role-plays
- `Writing/` — forms, messages and paragraphs
- `Dictation/` — sound-to-text work
- `Mixed-Revision/` — cumulative checkpoints
- `Answer-Keys/` — solutions and evaluation guides

Attempt before checking. Corrected work is more valuable than a high first-attempt score.
