# Week 8 Speaking Practice

**Milestone:** Complete realistic Goethe A1 tasks confidently and independently.

## Prompt ladder

1. Give five individual answers.
2. Connect the answers for 30 seconds.
3. Speak for the weekly target with keywords only.
4. Perform one question-and-answer exchange.
5. Change one key detail unexpectedly.
6. Use one repair phrase.

Useful patterns: `Bitte wiederholen Sie …`, `Die richtige Information ist …`, `Ich glaube, Antwort … ist richtig.`, `In der Nachricht steht …`, `Ich möchte …`, `Kann ich bitte …?`
