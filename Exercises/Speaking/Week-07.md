# Week 7 Speaking Practice

**Milestone:** Handle a simple health issue, appointment or emergency interaction.

## Prompt ladder

1. Give five individual answers.
2. Connect the answers for 30 seconds.
3. Speak for the weekly target with keywords only.
4. Perform one question-and-answer exchange.
5. Change one key detail unexpectedly.
6. Use one repair phrase.

Useful patterns: `Ich habe …`, `… tut mir weh.`, `Ich brauche einen Termin.`, `Rufen Sie bitte …`, `Ich habe … gemacht.`, `Können Sie das wiederholen?`
