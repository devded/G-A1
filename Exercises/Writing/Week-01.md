# Week 1 Writing Practice

Write 50–80 words appropriate to the topic **Sounds, Greetings and Identity**.

Required:

- include all four content points you choose,
- use at least three weekly words,
- include one time expression,
- include one question or request,
- use one connector,
- check verb position and noun capitals.

Create a corrected second version after using `Writing/SELF_CORRECTION_CHECKLIST.md`.
