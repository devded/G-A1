# Week 7 Grammar Drills

## A. Build statements

Use each pattern with true personal information:

1. `Ich habe …`
2. `… tut mir weh.`
3. `Ich brauche einen Termin.`
4. `Rufen Sie bitte …`
5. `Ich habe … gemacht.`
6. `Können Sie das wiederholen?`

## B. Word order

Reorder:

1. heute / ich / Deutsch / lerne
2. am Montag / wir / um acht Uhr / anfangen
3. nicht / sie / kommen / kann
4. einen Kaffee / möchte / er
5. du / wo / wohnst

## C. Transformation

1. Make one statement negative.
2. Turn one statement into a yes/no question.
3. Begin one statement with a time phrase.
4. Change the subject from `ich` to `wir`.
5. Create one polite `Sie` question.
